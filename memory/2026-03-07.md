### 飞书发本地文件的正确方式
- ❌ 错误：用 `path`、`filePath` 参数
- ❌ 错误：给飞书文档链接
- ❌ 错误：给 GitHub raw 链接
- ✅ 正确：用 `message` 工具的 `sendAttachment`，**参数必须是 `media`，值是本地文件的绝对路径**

示例：
```python
message(
    action="sendAttachment",
    media="/Users/suchuanlei/.openclaw/workspace/文件名.md"
)
```
